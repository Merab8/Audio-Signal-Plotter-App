# Audio Signal Plotter using MATLAB

## Overview
This project is an Audio Signal Plotter developed using MATLAB. The application allows users to record or load an audio signal and visualize its waveform in the time domain. It is designed to help understand basic audio signal behavior and signal representation through graphical plots.

## Features
- Record audio signal using MATLAB
- Load audio files for analysis
- Plot audio waveform in the time domain
- Simple and user-friendly implementation
- Useful for learning basic signal processing concepts

## Purpose
The main purpose of this project is educational. It helps students understand how audio signals are captured, processed, and visualized using MATLAB. This project is suitable for beginners learning signal processing and MATLAB programming.

## Technologies Used
- MATLAB
- MATLAB Audio and Signal Processing functions

## Applications
- Learning audio signal fundamentals
- Understanding time-domain signal representation
- Academic demonstrations and lab work

## How to Use
1. Open the project in MATLAB.
2. Run the main script or app file.
3. Record an audio signal or load an existing audio file.
4. View the plotted waveform of the audio signal.
